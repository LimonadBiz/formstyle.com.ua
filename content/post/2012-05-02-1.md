<span style="color: #000000;">Выполнив этот урок, вы создадите стильную форму контакта на свой сайт, используя свойства HTML 5 и CSS 3.0. Мы будем использовать новые возможности HTML 5, чтобы придать форме большей функциональности, а также новые свойства CSS 3.0, чтобы воплотить фантазии по красивой стилизации формы в реальность.</span>

<span style="color: #000000;">В дизайне формы мы будем использовать все методики, которые делают дизайн более запоминающимся. Это градиент, тени и блики. Для начала мы создадим концепт нашей будущей форме в Photoshop, а затем воссоздадим из файла PSD реальную форму, используя HTML 5 и CSS 3.0.<!--more--></span>

<span style="color: #000000;">Воссоздайте в фотошопе что-то подобное. Или даже можете поимпровизировать, и придумать свои градиенты для веб-формы. Примерная картинка приведена ниже.</span>

<a href="http://formstyle.com.ua/wp-content/uploads/2012/05/9-1.jpg" rel="lightbox[222]" title="9 (1)"><img class="aligncenter size-medium wp-image-224" title="9 (1)" src="http://formstyle.com.ua/wp-content/uploads/2012/05/9-1-300x288.jpg" alt="" width="300" height="288" srcset="http://formstyle.com.ua/wp-content/uploads/2012/05/9-1-300x288.jpg 300w, http://formstyle.com.ua/wp-content/uploads/2012/05/9-1.jpg 500w" sizes="(max-width: 300px) 100vw, 300px" /></a>

<span style="color: #000000;">Теперь, когда наглядный пример формы PSD готов, приступим к написанию кода HTML 5, чтобы сформировать каркас нашей формы. Следую принципам создания привлекательных веб-формы, которые перечислены в нашей статье</span> [7 советов по созданию привлекательных веб-форм][1]<span style="color: #000000;">, мы используем метки (labels) для удобства. Для каждого поля своя метка, которая отражает его назначение. К тому же, мы задействовали новый атрибут языка HTML 5 placeholder. Он позволяет выводить необходимое сообщение в каждом поле. Как только пользователь активирует поле, сообщение автоматически стирается. Очень удобная штука. Ранее такая фишка достигалась только при помощи JavaScript.</span>
